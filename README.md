# Personal Blog & Portfolio Website

A sleek, interactive blog and portfolio platform built with **Next.js**, **TypeScript**, and **Tailwind CSS**. Designed to showcase projects, achievements, and dynamic content with modern design patterns and smooth animations.

4. **SEO**: Update metadata in layout.tsx and individual pages
